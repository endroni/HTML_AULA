JavaScript externo

Uma forma prática de trabalhar com JavaScript é ...

A função fica em um arquivo externo. Isso facilita a Reutilização de código

Deixando o JS inline ou dentro da tag style, diminuem o número total de requisições HTTP e pode ser bom quando o código é pequeno. Porem, para códigos muito grandes, o recomendado mesmo é deixar externo.

Algumas vantagens!

  • Separação de HTML do código
  • Legibilidade
  • Velocidade no tempo de carregamento
  • Possibilidade de uso de mecanismos de cache do browser
